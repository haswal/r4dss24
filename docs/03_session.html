  4. Use arrange() to show how many cloudbuddies there are in cb_data.

  5. Many times when working with data, we need to use multiple different functions to achieve the desired end goal. One option is to create intermediate objects for each step of the analysis process, as can be seen below. A better alternative is to use the pipe %>% and link together different functions. Use the pipe method to convert the code below to a pipeline.

    cb_data_2 <- filter(cb_data,
                `Phase (color)` %in% c("1-lightgray",
                                       "6-red"
                                       ))
cb_data_3 <- select(cb_data_2, 
                    `Phase (color)`,
                    Volume)

cb_data_4 <- arrange(cb_data_3,
                     desc(Volume))

cb_data_5 <- rename(cb_data_4,
                    Phase = `Phase (color)`)

view(cb_data_5)
